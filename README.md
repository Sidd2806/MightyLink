MightyLink
MightyLink is a lightweight URL shortener built primarily with JavaScript. Designed for simplicity and ease of use, MightyLink lets you transform long URLs into short, shareable links with minimal setup.

Getting Started
Prerequisites
Node.js (Recommended v16+)
npm (comes with Node.js)
Installation
Clone the repo:

git clone https://github.com/Sidd2806/MightyLink.git
cd MightyLink
Install dependencies:

npm install
Usage
Start the server:

npm start
Open your browser and navigate to http://localhost:3000 (or configured port).

Shorten a link by entering the URL and clicking "Shorten".
